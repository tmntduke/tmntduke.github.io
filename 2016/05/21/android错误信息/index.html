android错误信息

android.view.WindowLeaked 窗体泄露

对话框、PopupWindown不能脱离Activity而单独存在着，当某个Dialog或者某个PopupWindow正在显示的时finish()了承载该Dialog(或PopupWindow)的Activity时，就会抛Window Leaked异常了，因为这个Dialog(或PopupWindow)的WindowManager已经没有依附的activity，所以它的窗体管理器已经泄漏了

  • 解决方法

    关闭(finish)某个Activity前，要确保附属在上面的Dialog或PopupWindow已经关闭(dismiss)了。

android.view.InflateException 布局文件加载错误

可能由布局文件中控件名称，属性名称等错误引起

或加载像素太大的图片 通常会伴随outOfMemory

  • 解决方法

    查找拼写错误的控件或属性名称

    加载大像素图片时使用 BitmapFactory.Options 进行图片的缩放**

java.lang.IllegalStateException:

The specified child already has a parent. You must call removeView() on the child’s parent first.

状态非法异常，在同一父容器中加载多次子控件
